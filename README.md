# Racing game

A Java racing game made with 2D concepts, that will contain laps, obstacles, score, two-player based, GUI.

## State:
- Work in progress

# Present functionalities:
- one player
- obstacles
- track
- illusion of movement, since the backround is moving, not the actual car
- controlling the car using up, down, left, right
- smooth turning animation

# Technologies:
- Java
- Swing/ Jpanel

# How to run project:
- clone repo
- open project
- run
